# Uploads Folder

This folder contains files that will be available for download from your website.

## CV / Resume

**To add your CV:**

1. Place your CV PDF file in this directory as `resume.pdf`
2. The file will be accessible at: `https://felixytang.github.io/academic-website/uploads/resume.pdf`
3. It's already linked in:
   - Navigation menu (CV link)
   - Author profile (Download CV button)
   - Homepage biography section

## Other Files

You can also upload:
- **Publication PDFs**: Name them descriptively (e.g., `deng2022-jgr-oceans.pdf`)
- **Poster PDFs**: Conference posters (e.g., `agu2022-poster.pdf`, `oceans2025-slides.pdf`)
- **Supplementary Materials**: Data, code, appendices

## File Naming Best Practices

- Use lowercase letters
- Replace spaces with hyphens (-)
- Include year and publication venue for papers
- Keep names short but descriptive
